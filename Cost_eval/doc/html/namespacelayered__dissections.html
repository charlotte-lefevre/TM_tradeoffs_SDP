Computes the costs of layered dissections Depending on the input, the applicable algorithm changes.

The main function to compute he times / memories, Skkl is computed using stirling approximation.

Parameters
Reminfloat minimum sub syndrome coefficient
Remaxfloat ditto with max
nbResint number of computed points
nbtargetint number of computed targets for memBlowUp
Returns
(m,t, col, res) with m / t log2 of memory / time cost relative to code length col indicates the corresponding regime for the point and res the target coefficient used. In other words m[i] t[i] is a time/memory cost with algorithm col[i] and l coefficient res[i]

Instanciated version of apply_calibration.

Parameters
Sdesired number of solutions. Here it is not necessarily equal to Skkl, as this can be used by memBlowup
tthe target coefficient such that target size is equal to l*t (so t = 1 is classical calibration)
Asymptoticif False computes memory / time cost in the binary metrics (asymptotic = True is useful when this algorithm is called by memBlowup - as the binary metrics conversion is done at the very end)
Returns
(m,t) log2 of memory / complexity
Attention
