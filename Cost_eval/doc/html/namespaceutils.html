Useful when we have plenty of memories and times and there are many points which are strictly better than others
(m,t) is strictly better than (m',t') if m<m' and t<t'.

Parameters
Mxthe array containing all memories
Tythe array containing all times so that M_x[i] is associated to T_y[i] for all i
colan optional array which is strongly associated to memory and times (ie col[i] is associated to M_x[i] and T_y[i])
Returns
epurated arrays (M_x, T_y) and if an array col is provided it also returns cols epurated
