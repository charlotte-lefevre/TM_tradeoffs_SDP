From a submatrix H_sub, it builds a list containing cardinal_list linear combinations of vectors if index is equal to 0, it takes into account the syndrom (that is populates the list with "linear_combinations - s") The result is returned as a dict "Linear combination ---> vector".

Attention
for practical reasons everything is returned as a matrix object (appending is much easier)
