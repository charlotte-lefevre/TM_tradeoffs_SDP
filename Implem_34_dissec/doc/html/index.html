Usage

In main.sage set nb_test to the desired number of runs, set verbose to True or False depending on the desired verbosity, and launch main.sage. The parity-checkmatrix and the syndrome are sampled uniformly at random

How to change the value of n, k and l

The value of is set to 560 because it gives a reasonnable runtime. To change it, one must change some parts of the code in the function layered_dissec in merges.sage which are made explicit in the comments.

Bottleneck for performances

It is sometimes quite difficult to control exactly the operations done in a language as high-level as sagemaths.

  • Sorting method: We use the built-in one (timsort). When sorting an element_list, it is done according to its value on linear_combin_trunc. In practise it is not the msot expensive part.
  • Most expansive part: in mergings, 60% of the time is spent creating new elements (function merge_new_element)
